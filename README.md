# Estratégia Educação — Páginas de Cursos

Este repositório contém as páginas HTML do site de divulgação de cursos da Estratégia Educação.

## Arquivos principais

- `index.html` — página principal (catálogo geral)
- `motoristas.html` — página para público de motoristas
- `mulheres.html` — página para público feminino
- `professores.html` — página para professores
- `tecnicos-por-competencia.html` — página de técnicos por competência
- `sindicos-administradoras-condominios.html` — página para síndicos/administradoras/condomínios

## Estrutura

Os arquivos estão na raiz do projeto (sem subpastas), prontos para publicação estática.

## Publicação

Você pode publicar este conteúdo em qualquer hospedagem de site estático (GitHub Pages, Netlify, Vercel, cPanel etc.).
